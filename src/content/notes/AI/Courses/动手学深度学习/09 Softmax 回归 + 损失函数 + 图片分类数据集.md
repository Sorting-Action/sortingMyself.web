# Softmax 回归

## 从回归到多类分类

其实是一个分类问题。

回归：单连续数值输出，损失是和真实的区别

分类：多个不同输出，输出的是置信度

### 过渡的方法：均方损失

假设有 N 个类别：
- 一位有效编码：$y = [y_1, y_2, ..., y_n]^T$
- 均方损失训练：$MSE=\frac{1}{n}\sum_{i=1}^n{\left( y_i-\hat{y}_i \right) ^2}$
- 最大值为预测：$\hat{y} = argmax_x o_i$

分类不关心实际值，**正确类的置信度必须非常大**

### 无校验比例

更置信的识别正确类（大余量）

$$o_y-o_i \geq \Delta (y, i)$$

### 校验比例

希望输出是一个概率，使用 Softmax。

$$\hat{y_i} = \frac{exp(o_i)}{\sum_kexp(o_k)}$$
- 所有的值非负，且和唯一
- 使用 exp 作非负
- 概率 $y$ 和 $\hat{y}$ 的区别作为损失

### 交叉熵损失

- 损失
- 梯度就是真实的差别
$$l(\textbf{y}, \hat{\textbf{y}})=-\sum_i y_ilog(\hat{y_i})$$
$$\partial_{o_i}l(\textbf{y}, \hat{\textbf{y}})=softmax(\textbf{o})_i-y_i$$

## 训练

输入是向量
- 展平图片，损失空间信息

## 总结

- 多分类模型
- Softmax 得到预测置信度
- 交叉熵判断损失

# 损失函数

预测和真实的区别

## L2 均方损失

$$ l(y, y') = \frac{1}{2} (y-y')^2$$

![](/notes-assets/752eb1b0/image-20240404211136662.png)

- 蓝色：$y=0$ 时预测值变化
- 绿色：似然函数高斯分布
- 橙色：梯度，高斯分布的梯度是线性的
	- 离原点远，梯度的绝对值大，更新的大

## L1 绝对值

离原点远的时候，不一定想要大的更新。

![](/notes-assets/752eb1b0/image-20240404211622227.png)

- 在负一和正一之间，梯度永远是常数，更新不会大，很稳健
- 预测和真实靠的近（预测末期）：不稳定，会有大的变动
