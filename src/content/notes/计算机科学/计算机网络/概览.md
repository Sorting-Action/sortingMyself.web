# 基本概念

端系统：接入网络的所有设备 end system

数据链路和分组交换机连接着端系统。当今常用的两种分组交换机分别是用于核心网的路由器和接入网的链路层交换机。端系统发出的信息包就是分组(pecket)，分组传输经历的路径是路径(path/route)。

端系统(end system)通过 ISP（因特网服务提供商）接入。每个 ISP 独立管理，运行 IP 协议，ISP 也有不同的层级，底层 ISP 由高层 ISP 连接，高层 ISP 之间直接互联。

交换机和端系统都运行着多个协议，他们控制信息的收发。TCP\\IP 协议是最重要的。

上面是从软件角度，下面从服务角度看：涉及多个相互交换数据的端系统就是分布式应用。他们通过套接字接口，socket，交付数据。

协议是通信的规则，说明了格式、顺序和需要的操作。

# 网络边缘

端系统，也就是主机，分为客户端和服务器。

对于家庭接入，有和电话、有线电视复用基础设施的分频复用技术，也有光纤到户。

企业和家庭：使用局域网技术，以太网是最常用的、WiFi（802.11）

广域：3G、4G、和 5G 信号

## 物理媒介

比特通过电磁波或光脉冲的方式跨越物理媒介。有导引式媒介（电缆、光缆）和非导引式媒介（空气、外层空间）。
