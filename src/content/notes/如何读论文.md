论文很长，读多遍

|            | 1    | 2    | 3   |
| ---------- | ---- | ---- | --- |
| title      | x    |      |     |
| abs        | x    |      |     |
| intro      |      | x    |     |
| method     | some | some |     |
| exp        | some | some |     |
| conclusion | x    |      |     |
|            |      |      |     |
1. 第一遍
	1. 判断自己是不是需要继续读下去
2. 第二遍
	1. 看看 intro 里面重要的文献，有没有读过，如果不懂，先看他们。
	2. 知道细节
	3. 了解优秀研究者的世界观，观点（要综合很多研究者的观点，总结出属于自己的观点）
3. 第三遍
	1. 详细的方法读
	2. 自己的实现
	3. 我需要怎么做
	4. 未来能怎么做
