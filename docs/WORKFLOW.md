# AI 协作工作流日志

> 本项目的隐藏目标：练习并迭代「人与 AI 协作」的工作流。
> 规则：**每次工作会话结束，由人亲手写下面的五行日志**。这不是 AI 生成的内容，是练手的产出物本身。

## 日志模板

```
### YYYY-MM-DD
- 我问了 AI 什么：
- AI 做对了什么：
- AI 做错了什么 / 我纠正了什么：
- 下次怎么问更好：
- 新发现的技巧：
```
